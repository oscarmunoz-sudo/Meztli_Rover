# Meztli_Rover

Paquetes de sensores y brazo robótico sobre **ROS 2** con soporte de simulación en **RViz**.  
Este repositorio concentra el desarrollo y pruebas de software para integración en prototipos reales del rover.

## 📌 Descripción

**Meztli_Rover** contiene programas orientados a:

- Integración de sensores
- Control e interacción con brazo robótico
- Flujo de trabajo con ROS 2
- Visualización y validación en RViz

> Estado actual: documentación e imágenes iniciales de prototipos reales.  
> Próximamente se añadirán más recursos de simulación (RViz/entorno virtual).
